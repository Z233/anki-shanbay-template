# 扇贝单词 Anki 模版

## 📃 这是啥子

一个高仿扇贝单词界面的 Anki 词汇模版，目前只能在 Ankidroid 上使用。

## 🍽 如何食用

下载 apkg 目录下的 anki-template-shanbay.apkg 导入 Anki 即可。
