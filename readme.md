# 扇贝单词 Anki 模版

## 📃 这是啥子

一个高仿扇贝单词界面的 Anki 词汇模版，目前只能在 Ankidroid 上使用。

## 🍽 如何食用

1. 下载 apkg 目录下的 anki-template-shanbay.apkg 导入 Ankidroid
2. Ankidroid 中，设置 > 手势操作，启动手势并把所有的动作设为无
3. Ankidroid 中，设置 > 复习，把全屏显示一项设为隐藏系统状态栏和回答按钮
